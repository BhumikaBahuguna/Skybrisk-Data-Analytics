# Skybrisk Data Analytics Internship

This repository contains a structured, week-by-week data analytics internship project designed to build strong foundations in Python, data structures, data analysis, and exploratory data analysis (EDA).

## Project Overview

The internship is organized into guided phases so each topic can be completed in a clean, professional, and portfolio-ready way.

### Month 1: Python and Core Data Analytics

- Week 1: Python basics
- Week 2: Data structures
- Week 3: NumPy and Pandas
- Week 4: Data visualization

### Applied EDA Projects

- Task 3: Customer churn exploratory data analysis
- Task 4: Netflix exploratory data analysis

## Objectives

- Build clean and reusable Python programs
- Practice problem solving with core data structures
- Learn array and dataframe operations using NumPy and Pandas
- Create meaningful visualizations with Matplotlib and Seaborn
- Develop professional EDA workflows for real-world datasets

## Getting Started

1. Create and activate a virtual environment.
2. Install the project dependencies:

```bash
pip install -r requirements.txt
```

3. Complete the project phase by phase.
